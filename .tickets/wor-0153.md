# Implement remote/API WordSource backend

Create src/backend_remote.go in the root package. Implement remoteWordSource, an unexported struct satisfying WordSource that performs reverse lookup via HTTP GET against a conforming API endpoint.

## Spec References
- docs/specs/wordlist-backends.md §"Remote / API Source" and §"Suggestions"
- docs/specs/code-architecture.md §3.6 (WordSource interface)

## Implementation Details

### API Contract

GET {baseURL}/words?value={value}&system={system}&limit={limit}

Success response (200 OK):
  {
    "words": [
      {"hebrew": "שלום", "transliteration": "shalom", "meaning": "peace"}
    ],
    "hasMore": false
  }

word fields: hebrew required; transliteration and meaning optional (absent fields treated as empty string).

Non-200 response: return error with format "remote word source: HTTP {statusCode} from {url}".
Network failure: return wrapped error with fmt.Errorf("remote word source: %w", err).

### Unexported Struct

  type remoteWordSource struct {
    baseURL    string
    httpClient *http.Client
    authToken  string // Bearer token; empty means unauthenticated
  }

Compile-time check:
  var _ WordSource = (*remoteWordSource)(nil)

remoteWordSource does NOT implement io.Closer (HTTP client requires no cleanup).

### Functional Options Pattern

  type RemoteOption func(*remoteWordSource)

  func WithAuthToken(token string) RemoteOption
  func WithHTTPClient(client *http.Client) RemoteOption

### Exported Constructor

  // NewRemoteWordSource creates a WordSource backed by an HTTP API.
  // baseURL must begin with "http://" or "https://".
  // Use WithAuthToken to supply a Bearer token for authenticated endpoints.
  func NewRemoteWordSource(baseURL string, opts ...RemoteOption) (WordSource, error)

Validates that baseURL is non-empty and begins with http:// or https://. Returns an error for invalid baseURL. Uses http.DefaultClient when no WithHTTPClient option is provided.

### FindByValue

1. Construct URL: net/url.Parse(baseURL + "/words"), set query params value, system, limit.
2. Build *http.Request with method GET.
3. If authToken != "": set header Authorization: Bearer {token}.
4. Execute request via httpClient.Do(req).
5. Read and decode JSON body (limit to 1 MiB with io.LimitReader to prevent memory exhaustion from malicious servers).
6. Map response words to []gematria.Word.
7. Return (words, hasMore, nil) on success.

### Root Package Import Constraint

src/backend_remote.go may import net/http, net/url, encoding/json, io, fmt, and strconv. It must NOT import "os" or any internal/cli package.
