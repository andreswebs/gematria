# Implement SQLite WordSource backend

Create src/backend_sqlite.go in the root package. Implement sqliteWordSource, an unexported struct that satisfies both WordSource and io.Closer by querying a pre-indexed SQLite database generated by the gematria index subcommand (index-subcommand task).

## Spec References
- docs/specs/wordlist-backends.md §"Embedded Database" and §"Suggestions"
- docs/specs/code-architecture.md §3.6 (WordSource interface)

## Implementation Details

Dependency: add modernc.org/sqlite to src/go.mod (CGo-free pure-Go driver — no external C library required).

### Database Schema (created by gematria index, read here)

  CREATE TABLE words (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    hebrew TEXT NOT NULL,
    transliteration TEXT NOT NULL DEFAULT '',
    meaning TEXT NOT NULL DEFAULT ''
  );
  CREATE TABLE word_values (
    word_id INTEGER NOT NULL REFERENCES words(id),
    system TEXT NOT NULL,
    value INTEGER NOT NULL
  );
  CREATE INDEX idx_word_values ON word_values(system, value);

### Unexported Struct

  type sqliteWordSource struct { db *sql.DB }

Compile-time checks:
  var _ gematria.WordSource = (*sqliteWordSource)(nil)
  var _ io.Closer           = (*sqliteWordSource)(nil)

### Exported Constructor (hides implementation per Effective Go)

  // OpenSQLiteWordSource opens an existing SQLite word-list database.
  // The database must have been produced by "gematria index --format sqlite".
  // Callers should check whether the returned WordSource also implements
  // io.Closer and call Close when done.
  func OpenSQLiteWordSource(path string) (WordSource, error)

- Opens DB with sql.Open("sqlite", path)
- Runs a lightweight schema verification (PRAGMA table_info) to return a clear error for wrong-format files
- Returns the interface value (not the concrete type)

### FindByValue

  SELECT w.hebrew, w.transliteration, w.meaning
  FROM words w
  JOIN word_values v ON v.word_id = w.id
  WHERE v.system = ? AND v.value = ?
  LIMIT ?

Query with limit+1 rows. Collect rows into []Word. Return results[:min(len,limit)], len(results)>limit, nil.
System argument must be converted to its string representation (e.g., "hechrachi") for the SQL parameter.

### Close

Delegates to db.Close(). Returns any error from the driver.

### Root Package Import Constraint

src/backend_sqlite.go may import database/sql, modernc.org/sqlite, and standard library packages. It must NOT import "os", "net/http", or any internal/cli package.
